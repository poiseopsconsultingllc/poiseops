PoiseOps Pages Pack
===================

What this adds
- /services/ (plus 5 service detail pages)
- /industries/ (plus 4 industry pages)
- /about/, /contact/, /privacy/, /terms/
- styles.css (shared styling for the new pages)
- sitemap.xml (updated to include these new pages)

How to install (GitHub Pages + Cloudflare)
1) Download this zip and unzip it.
2) Upload the folders (services/, industries/, about/, contact/, privacy/, terms/) into your repo root.
3) Upload styles.css into your repo root.
4) Replace your existing sitemap.xml with the sitemap.xml included here.
5) Commit + push. GitHub Pages will publish automatically.

Notes
- Your existing index.html is NOT modified by this pack.
- Once deployed, the new pages will be crawlable via /sitemap.xml.
- Next optional step: add visible links from the homepage nav/sections to these pages.
